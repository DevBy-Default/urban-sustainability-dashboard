# Neural City Urban Sustainability Dashboard

A prototype dashboard built for the Neural City Product Engineering Internship Assignment.

## Features

- AQI Comparison
- Green Cover Analysis
- Population Density Comparison
- Urban Sustainability Index
- City Rankings
- AI Insights
- Responsive Dashboard

## Data Sources

- CPCB Air Quality Data
- Census of India
- Forest Survey of India

## Deployment

Built using React, TypeScript, Tailwind CSS and Recharts.

## Purpose

To help municipal officers, urban planners and citizens compare Indian cities using sustainability metrics.
